# Framer.com -- Homepage Design Analysis

> Research conducted April 2026. Data gathered via web search, design analysis sources, and publicly available information. Direct page fetch was blocked (403).

---

## SEO Metadata

- **Title tag**: "Framer: Create a professional website, free. No code website builder loved by designers."
- **Meta description**: "Build a free website with Framer -- enjoy full design freedom, powerful CMS, built-in SEO, and real-time collaboration. Create professional, fully custom sites with the no-code builder loved by designers and high-performing teams."

---

## Hero Headline

The hero section communicates Framer's core positioning as a no-code website builder. Based on available data, the primary hero messaging centers on:

> **"Create a professional website, free."**

With supporting copy emphasizing: full design freedom, powerful CMS, built-in SEO, real-time collaboration, no code required.

### CTA Buttons (Hero)

- **"Start for free"** -- primary CTA (dark button, prominent placement)
- Secondary links to features, pricing, and templates

---

## Section Structure (Top to Bottom)

1. **Navigation bar** -- Logo, product links (Features, Templates, Pricing, Enterprise, Resources), CTA "Start for free"
2. **Hero section** -- Main headline, subheadline describing key capabilities, primary CTA, animated product showcase/demo
3. **Product showcase / Design canvas** -- Interactive demonstration of the Framer editor, showing the visual design experience
4. **Features grid** -- Key capabilities: Design Freedom, CMS, SEO Tools, Animations, Responsive Design, AI Generation
5. **AI Website Generation** -- Framer AI feature showcase: generate websites from text prompts
6. **CMS / Content Management** -- Dynamic content, collections, filtering
7. **SEO & Performance** -- Built-in optimization tools, Core Web Vitals, sitemap generation
8. **Animations & Interactions** -- Scroll animations, hover effects, transitions, visual effects
9. **Templates gallery** -- Curated template showcase with categories
10. **Social proof / Logos** -- Trusted by companies, gallery of sites built with Framer
11. **Pricing section** -- Free, Basic ($5/mo), Pro ($15/mo), Scale, Enterprise tiers (simplified in late 2025)
12. **Footer CTA** -- Final conversion block with "Start for free"
13. **Footer** -- Links, legal, social

## Color Palette

Framer's own website uses a **light-dominant theme** with dark mode support:

| Role | Color | Hex |
|------|-------|-----|
| Background (light) | White | `#FFFFFF` |
| Background (dark) | Near-black | `#0D0D0D` |
| Primary text | Dark/Black | `#000000` / `#171717` |
| Secondary text | Gray | `#666666` |
| Accent / Brand | Blue-violet | `#0099FF` (Framer blue) |
| CTA background | Black | `#000000` |
| CTA text | White | `#FFFFFF` |
| Borders / Dividers | Light gray | `#E5E5E5` |
| Subtle backgrounds | Off-white | `#F5F5F5` |

Framer's brand color is a distinctive **bright blue** that appears in the logo and accent elements. The overall palette is minimal and high-contrast.

## Typography

| Element | Font | Weight | Notes |
|---------|------|--------|-------|
| Primary / Body | **Inter** | 400, 500 | Inter 4.0 is Framer's default; used across the platform |
| Headlines | **Inter Display** | 600, 700 | Display optical size variant for large text |
| Monospace (code) | System monospace | 400 | For code snippets and technical elements |

- **Heading sizes**: Hero ~56-72px, Section headings ~36-48px, Sub-headings ~24-28px
- **Body text**: 16-18px
- **Line height**: 1.3-1.5 for body, tighter for headlines
- Inter version 4.0 supports all OpenType features including ligatures and alternates

## Visual Effects

- **Scroll animations**: Elements fade in, scale, and translate on scroll. Framer's own site demonstrates its animation engine
- **Hover interactions**: Subtle scale and opacity transitions on cards and interactive elements
- **Smooth transitions**: Page-level transitions between sections
- **Responsive breakpoints**: Fluid layouts that adapt across desktop, tablet, mobile
- **Video/animated demos**: Product interface demonstrations showing the editor in action
- **Gradient accents**: Subtle gradient overlays on certain sections
- **No heavy 3D/WebGL/particles**: The site favors clean, performance-focused design over heavy visual effects

---

## Image/Video Style

- **Product screenshots**: Real UI of the Framer editor, showing actual design workflows
- **Template previews**: High-quality rendered previews of available templates
- **Clean mockups**: Device frames showing responsive designs
- **Minimal illustration**: Very little abstract illustration; focus is on actual product UI
- **Video demos**: Short looping videos demonstrating features like drag-and-drop, animations

## Tone of Voice

- **Accessible and confident**: Not overly technical, not dumbed down
- **Designer-centric**: Speaks to designers who want creative control without code
- **Empowering**: "Create", "Design", "Build", "Publish" -- action-oriented verbs
- **Clean and direct**: Short sentences, clear value propositions
- **No jargon**: Avoids developer-heavy terminology; positions itself as a creative tool
- **Premium but approachable**: Feels professional without being corporate

---

## How They Talk About AI (Without Cliches)

Framer positions AI as a **practical tool**, not a futuristic concept:

- "Generate your website from a text prompt" -- describes the action, not the technology
- AI is presented as one feature among many, not the entire product identity
- No "revolutionary", no robots, no neon, no sci-fi imagery
- The AI feature is shown through actual product UI, not abstract visualizations
- Emphasis on the **output** (a professional website) rather than the **mechanism** (artificial intelligence)
- Language pattern: **"AI helps you [specific action]"** rather than "AI transforms everything"

---

## Key Takeaways for FlowAI

1. **Clean, minimal aesthetic** with high contrast and generous white space
2. **Product-first storytelling**: Show the actual tool in action
3. **Inter font family** provides excellent readability and modern feel
4. **CTA simplicity**: One primary action repeated consistently
5. **Feature-benefit structure**: Each section pairs a capability with its outcome
6. **Light theme default** with dark mode support -- considers both preferences
7. **AI as utility, not identity**: Integrates AI naturally without making it the entire brand
