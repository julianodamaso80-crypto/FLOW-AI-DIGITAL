7. **How It Works** - The prompt-to-deploy workflow
8. **Model Selection** - Multiple AI model options (Sonnet 4.6, etc.)

10. **Blog/Updates Section** - Links to latest features

## 10. How They Talk About AI (Without Cliches)

Bolt.new's approach to AI language:

- **"Using your words"** - AI responds to natural language, positioned as natural communication
- **"Prompt, run, edit, and deploy"** - Four concrete action verbs, no AI mystification
- **"AI agents write the code, set up the project structure, install dependencies, and deploy"** - Describes exactly what the AI does in practical terms
- **"AI-powered development platform"** - Simple descriptor, not hyperbolic
- **"Frontier coding agents from the AI labs"** - Technical but grounded
- **"How can Bolt help you today?"** - Service-oriented prompt, like a helpful assistant
- **"AI prompt enhancement"** - Practical feature description
- **"Build production-grade full-stack applications with ease"** - Focuses on output quality, not AI intelligence
- **"The best balance of speed, cost, and reasoning depth"** (about model selection) - Technical, honest trade-off language

**Key pattern**: Bolt positions AI as a practical tool that does specific things (writes code, deploys apps) rather than making grand claims about intelligence or revolution. The focus is on the user's workflow (prompt, run, edit, deploy) and the quality of the output.

---

## 11. Key Design Takeaways for FlowAI

1. **Prompt-as-hero** - The input box IS the homepage hero, zero friction to start
2. **Dark, developer-aesthetic** - Professional dark theme with blue accents
3. **Four-verb workflow** - "Prompt, run, edit, deploy" is brilliantly simple messaging
4. **Inclusive audience** - Explicitly addresses developers, PMs, AND designers
5. **Scale proof** - "Millions" and "1 million websites in five months" builds credibility
6. **AI as helper, not genius** - "How can Bolt help you today?" is service language
7. **Design system awareness** - Homepage includes design system selection, showing sophistication
8. **Real-time feedback** - Live preview builds trust and excitement
9. **StackBlitz WebContainer** - Browser-native technology (no server setup) is a key differentiator
10. **Model transparency** - Openly mentions which AI model is being used (Sonnet 4.6), building trust through technical transparency
