## 4.3 Visualizacao dos 129 Agentes — 3 Propostas

### Proposta A: Constellation Orbital (Recomendada)

**Referencia**: Inspirado em mistral.ai, versao FlowAI
**Conceito**: Centro com logo FlowAI. 12 nodes grandes em circulo (squads). Linhas conectando tudo. Hover em squad revela seus agentes. Pulso animado no centro.
**Dificuldade**: Media. Canvas 2D, ~100 linhas de JS.
**Vantagem**: Visual impressionante, interativo, leve.

### Proposta B: Grid de Cards Expandiveis

**Referencia**: Inspirado em huggingface.co
**Conceito**: Grid 4x3 com cards dos 12 squads. Click expande card mostrando os agentes daquele squad. Counter "129 agentes" acima.
**Dificuldade**: Baixa. HTML/CSS puro com JS para toggle.
**Vantagem**: Simples, acessivel, mobile-friendly.

### Proposta C: Network Graph Interativo

**Referencia**: Inspirado em visualizacoes de rede neural
**Conceito**: Todos os 129 agentes como pontos pequenos, organizados em clusters por squad. Linhas conectam agentes do mesmo squad. Linhas mais finas conectam squads entre si. Zoom e pan.
**Dificuldade**: Alta. Precisa de Canvas ou SVG complexo.
**Vantagem**: "Wow factor" maximo.

### Recomendacao Final

**Proposta A (Constellation)** para desktop + **Proposta B (Grid)** como fallback mobile. Melhor relacao impacto vs complexidade.

```
Lead → Trafego → Tracking → CRM → Atendimento → Follow-up → Vendas → Dashboard
  ↑                                                                         |
  └─────────────────── Remarketing ←────────────────────────────────────────┘
```

### Implementacao Tecnica

- SVG com 8 nodes posicionados em circulo/oval
- Path conectando todos na sequencia
- `stroke-dasharray` + `stroke-dashoffset` animado para efeito de "dado fluindo"
- Labels em cada node
- Pulso (scale) no node ativo
- Velocidade: 1 ciclo completo a cada 8 segundos

## 4.7 Stack Tecnica

- **HTML estatico + CSS puro + JS vanilla** (sem framework, sem build step)
- **Lenis** (~8KB) para smooth scroll — via CDN
- **GSAP** (~30KB) opcional para animacoes mais complexas — via CDN
- **Three.js** NAO recomendado (150KB). Usar Canvas 2D para constellation.
- **Deploy**: Dockerfile (nginx:alpine) → GitHub Actions → EasyPanel

## 4.8 Performance

| Metrica | Target | Estrategia |
|---------|--------|------------|
| Lighthouse | 95+ | HTML semantico, CSS otimizado, JS defer |
| LCP | < 2.5s | Sem video hero (usar CSS gradient mesh), preload fonts |
| CLS | < 0.1 | Dimensoes explicitas em imagens, font-display swap |
| Peso total | < 2MB | Sem video 20MB, imagens WebP, lazy load |
| Fonts | Rapido | Preconnect Google Fonts, display=swap |
| JS | Nao bloqueia | defer em todos os scripts |
