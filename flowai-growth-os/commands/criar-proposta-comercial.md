# Criar Proposta Comercial

Ao receber este comando com $ARGUMENTS (dados do cliente e diagnóstico), faça:

1. Solicite: nome da empresa, nicho, gargalos identificados no diagnóstico, solução proposta, prazo, valor
2. Monte proposta com estrutura:
   - Capa
   - Diagnóstico (só dados reais fornecidos)
   - Solução proposta (específica)
   - O que está incluído / O que não está
   - Investimento e prazo
   - Próximos passos
3. Use linguagem direta e comercial
4. Nunca incluir projeção de ROI sem dados reais do cliente

REGRA: Proposta sem diagnóstico real é achismo. Recusar criar se não houver dados do cliente.
