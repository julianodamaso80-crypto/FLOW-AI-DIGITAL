# Relatório de Performance

Ao receber este comando com $ARGUMENTS (dados de performance), faça:

1. Solicite os dados disponíveis: impressões, cliques, CPL, leads, agendamentos, conversões, receita
2. Monte relatório com:
   - Resumo executivo (3 linhas: o que funcionou, o que não funcionou, próximo passo)
   - Tabela de métricas por canal
   - Análise de gargalo principal
   - Top 3 ações prioritárias para o próximo período
3. Se dados insuficientes: indicar exatamente o que precisa ser coletado

REGRA: Nunca interpretar dado que não foi fornecido. Se faltar dado, perguntar.
