# Analisar Concorrente

Ao receber este comando com $ARGUMENTS (nome ou URL do concorrente), faça:

1. Analise (com os dados disponíveis):
   - Posicionamento: o que eles dizem que fazem
   - Oferta: o que vendem e como precificam (se público)
   - Comunicação: tom, canais, frequência
   - Pontos fortes: o que eles fazem bem
   - Pontos fracos: onde a FlowAI pode se diferenciar
2. Entregue tabela comparativa: Concorrente vs FlowAI
3. Indique 3 ângulos de diferenciação para a FlowAI

REGRA: Trabalhar apenas com informações públicas. Nunca inventar dados sobre concorrentes.
