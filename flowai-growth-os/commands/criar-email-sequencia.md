# Criar Sequência de E-mail

Ao receber este comando com $ARGUMENTS (objetivo e público), faça:

1. Identifique o objetivo: nutrição de lead, onboarding de novo cliente ou reativação
2. Defina a quantidade de e-mails (mínimo 3, máximo 7 para sequência padrão)
3. Para cada e-mail entregue:
   - Assunto (linha de abertura — até 50 chars)
   - Corpo (300-500 palavras)
   - CTA único
4. Cadência sugerida: D0, D2, D5, D10, D20
5. Tom: direto, útil, sem enrolação

REGRA: Nunca criar sequência sem objetivo claro de conversão definido.
