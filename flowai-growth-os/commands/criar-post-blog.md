# Criar Post Blog

Ao receber este comando com $ARGUMENTS (tema ou palavra-chave), faça:

1. Confirme a palavra-chave principal
2. Estruture o artigo com:
   - Título (H1) com a palavra-chave
   - Introdução que valida a dor do leitor (150-200 palavras)
   - 3 a 5 seções com H2s
   - Conclusão com CTA para Raio-X de Receita
3. Extensão: 800 a 1.500 palavras
4. Tom: especialista que fala com dono de negócio, não com profissional de marketing
5. Sempre terminar com link para agendamento do Raio-X de Receita

REGRA: Nunca inventar dados ou estudos. Se usar dado, marcar como [VERIFICAR FONTE].
