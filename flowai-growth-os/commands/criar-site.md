# Criar Site ou Landing Page

Ao receber este comando com $ARGUMENTS (briefing do projeto), faça:

1. Pergunte se está criando para a FlowAI ou para um cliente
2. Se for cliente: peça nicho, oferta principal, público, diferencial e CTA desejado
3. Use a skill site-e-landing-page como base
4. Entregue estrutura completa: headline, subheadline, seção de problema, solução, prova, oferta, CTA, FAQ
5. Indique claramente onde inserir imagens, depoimentos e cases reais
6. Nunca invente provas sociais — marque como [INSERIR DEPOIMENTO REAL]

REGRA: Nunca criar landing page sem oferta e CTA definidos.
