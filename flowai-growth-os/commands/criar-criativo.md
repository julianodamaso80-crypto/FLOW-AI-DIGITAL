# Criar Criativo

Ao receber este comando com $ARGUMENTS (briefing do criativo), faça:

1. Identifique: nicho, procedimento/serviço, público, objetivo (clique, lead, ligação)
2. Crie 3 versões com ângulos diferentes:
   - Versão A: ângulo dor/problema
   - Versão B: ângulo resultado/transformação
   - Versão C: ângulo prova/depoimento (marcar como [INSERIR PROVA REAL] se não fornecida)
3. Para cada versão entregue: título (até 40 chars), corpo (até 150 chars), CTA, sugestão de visual
4. Indique qual versão testar primeiro e por quê

REGRA: Nunca usar estatísticas ou dados sem fonte fornecida.
