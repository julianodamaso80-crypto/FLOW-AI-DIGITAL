# Criar Post Instagram

Ao receber este comando com $ARGUMENTS (tema ou briefing), faça:

1. Pergunte o formato desejado se não especificado: carrossel, estático ou reels
2. Pergunte o objetivo: educar, posicionar, converter ou engajar
3. Use os pilares de conteúdo da FlowAI como base
4. Entregue:
   - Carrossel: copy de cada slide (mínimo 5, máximo 10) + sugestão visual
   - Estático: copy + legenda + hashtags
   - Reels: roteiro por tempo (0s, 5s, 15s, 25s, CTA final)
5. Sempre incluir CTA no final

REGRA: Todo post deve ter objetivo de funil definido — nunca conteúdo de vaidade.
