# Skill: Tráfego Pago

## Regra antes de qualquer campanha

Nunca recomendar tráfego pago sem antes verificar:
1. Existe oferta clara e específica?
2. Existe página de destino com CTA definido?
3. Existe processo de atendimento para receber o lead?
4. Existe follow-up estruturado?

Se qualquer um desses itens faltar, a primeira ação é corrigir isso — não subir anúncio.

## Estrutura de campanha para os nichos

**Clínicas (odonto e estética)**
- Objetivo: geração de leads (formulário ou WhatsApp)
- Público: raio geográfico 5-15km, faixa etária conforme procedimento
- Criativo: antes/depois ou depoimento real + oferta específica de procedimento
- Landing page: simples, com CTA para WhatsApp ou formulário
- Remarketing: quem visitou a página mas não converteu

**Imobiliárias**
- Objetivo: geração de leads qualificados
- Público: intenção de compra/aluguel por bairro e faixa de renda
- Criativo: imóvel específico com diferenciais + CTA para conversa com corretor
- Landing page: formulário + mapa + fotos + CTA
- Remarketing: quem viu mas não preencheu

## Estrutura de verba enxuta (R$ 30-100/dia)

- 70% para campanha principal (conversão)
- 20% para remarketing
- 10% para teste de novo criativo

## Processo de otimização
- Semana 1: rodar 3 criativos diferentes
- Semana 2: pausar os 2 piores, escalar o melhor
- Semana 3: testar novo ângulo no criativo vencedor
- Semana 4: relatório de CPL, taxa de agendamento, custo por venda

## REGRA ABSOLUTA
Nunca projetar CPL, ROI ou faturamento sem dados reais da campanha anterior.
Se não há histórico, apresentar o plano e indicar quais métricas acompanhar.
